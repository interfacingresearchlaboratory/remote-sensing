KCL Database of Small Dams in the Volta (Unsnapped)
 
This is a database of small dams in the Volta basin developed by digitising visible dams in Google Earth. 
 
CITATION
Smedley, D. (2016)  A comprehensive database of dams for the Volta. www.globaldamwatch.org

 
TO USE THE DATA
To access the files content of the zip file must be uncompressed in a single folder. The data are stored as a Shapefile in geographic coordinates of decimal degrees based on the World Geodetic System, 1984 (WGS84).
 
To view these data in ArcMap, simply unzip to an accessible directory and open the Shape File.
